Este repositório foi criado para práticas e anotações de comandos e instruções no sistema git.

Leia o arquivo "Comandos e instruções" caso procure por informações sobre configurações ou 
sobre algum dos comandos estudados.

Instruções para colaboração em projetos, repositórios remotos e servidores em rede local estão disponíveis!

Padronização: 

Aspas simples: quando encontradas aspas simples em um comando, escreva-o sem aspas. Exemplo: 
git checkout 'nome da branch' => git checkout branch_nome

Aspas duplas: quando encontradas aspas duplas em um comando, mantenha-as. Exemplo: 
git commit -m "mensagem" => git commit -m "Inicia Repositório"
